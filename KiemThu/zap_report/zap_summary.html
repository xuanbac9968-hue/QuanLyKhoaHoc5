Content Security Policy (CSP) Header Not Set Medium CWE-693 Thiếu header CSP — trình duyệt không biết nguồn tài nguyên hợp lệ, tăng nguy cơ XSS và data injection. Thêm middleware CSP vào Program.cs:
app.Use(async (ctx, next) => { ctx.Response.Headers["Content-Security-Policy"] = "default-src 'self'; script-src 'self' cdn.jsdelivr.net; style-src 'self' cdn.jsdelivr.net 'unsafe-inline'"; await next(); });
Missing Anti-clickjacking Header Medium CWE-1021 Thiếu X-Frame-Options hoặc CSP frame-ancestors — trang có thể bị nhúng trong iframe để thực hiện tấn công clickjacking. Thêm header X-Frame-Options:
ctx.Response.Headers["X-Frame-Options"] = "SAMEORIGIN";
Sub Resource Integrity (SRI) Missing Medium CWE-345 Bootstrap Icons được load từ CDN (cdn.jsdelivr.net) không có thuộc tính integrity — nếu CDN bị tấn công, có thể inject mã độc. Thêm integrity + crossorigin vào thẻ link:
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.3/font/bootstrap-icons.min.css" integrity="sha384-tViUnnbYAV00GntAiPulivAtGs2ZbRFEFejJ4qLfBRIC7XKiiJjkfFTdOnBPMNK" crossorigin="anonymous" />

ctx.Response.Headers["X-Content-Type-Options"] = "nosniff";

✅ Điểm Bảo Mật Tốt

🛡 CSRF Protection (Anti-Forgery Token)

ZAP quét form đăng nhập — phát hiện __RequestVerificationToken trên mọi POST request. Không tìm được lỗ hổng CSRF nào.

🛡 Không có SQL Injection

Active scan thử nhiều payload SQLi trên form đăng nhập (Email, MatKhau) — không tìm được lỗ hổng. EF Core + parameterized queries hoạt động tốt.

🛡 Không có Reflected XSS

Razor Pages tự động HTML-encode output — không tìm được XSS reflected trên các trang public.

🛡 Authentication đúng chuẩn

Tất cả trang admin (/Admin/*, /KhoaHoc/*) đều redirect về login khi chưa auth — không lộ dữ liệu.

🛡 Không có Directory Listing

Không tìm được trang liệt kê file/thư mục — cấu hình StaticFiles an toàn.
🔧 Hướng Dẫn Khắc Phục (Program.cs)
⚡ Tất cả 4 vấn đề tìm được đều có thể fix bằng cách thêm một middleware duy nhất vào Program.cs

Thêm đoạn code sau vào Program.cs (sau app.UseStaticFiles()):

// Security Headers Middleware app.Use(async (ctx, next) => { // Chống Clickjacking ctx.Response.Headers["X-Frame-Options"] = "SAMEORIGIN"; // Chống MIME sniffing ctx.Response.Headers["X-Content-Type-Options"] = "nosniff"; // Content Security Policy ctx.Response.Headers["Content-Security-Policy"] = "default-src 'self'; " + "script-src 'self' cdn.jsdelivr.net 'unsafe-inline'; " + "style-src 'self' cdn.jsdelivr.net 'unsafe-inline'; " + "img-src 'self' data:; " + "font-src 'self' cdn.jsdelivr.net;"; // Referrer Policy ctx.Response.Headers["Referrer-Policy"] = "strict-origin-when-cross-origin"; await next(); });

Fix SRI cho Bootstrap Icons trong _Layout.cshtml:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.3/font/bootstrap-icons.min.css" integrity="sha384-tViUnnbYAV00GntAiPulivAtGs2ZbRFEFejJ4qLfBRIC7XKiiJjkfFTdOnBPMNK" crossorigin="anonymous" />
